# ATS Resume Scanner
## This is a resume checking application
## Follow these steps to check this out
### Step 1:- Simply download or clone this repository
### Step 2:- Put all your resumes inside the resumes folder (not inside resume/report)
### Step 3:- Run 'Resume Scan.exe'
### Step 4:- A simple UI will appear where you will be entering your keywords which will be used to analyze the provided resumes.
### Finally, the analysis will start once you press on 'Save List and Start Analysis' and a report will be created as 'report.csv' inside the 'resumes/report' directory.
